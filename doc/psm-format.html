Peptide-Spectrum Match (PSM) Format

Description

Some MEME Suite programs allow as input a file containing tab-delimited data from analysis of tandem mass spectrometry. Some popoular softwares that produce files that are supported by the MEME Suite include Crux's Tide and Percolator, Comet, and MS-GF+, and tab-delimited exports from TPP's PepXML Viewer.
